[의존객체 자동 주입]
- 스프링 컨테이너에 
생성자 (<constructor-org>), 
setter (<property>) 태그로
필요한 객체를 직접 주입

- 직접 주입하지 않고 스프링 컨테이너가 알아서 자동 주입하도록 함

1) @Autowired (== @Inject, 거의 안씀)
: 자동으로 주입
: 객체 타입이 일치하는 객체 주입
: 생성자, 필드, 메소드 사용 가능
: Spring 제공

2) @Resource
: 자동으로 주입
: 객체 이름이 일치하는 객체 주입
: 필드, 메소드 사용 가능
: Java 제공
